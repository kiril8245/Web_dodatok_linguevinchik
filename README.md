# Лабораторна робота №1
## Тема: Розробка статичного інтерфейсу Web-додатка 

### Інформація про студента
* **ПІБ:** Гудима Кирило Олегович
* **Група:** КВ-34

### Завдання до роботи 
Розробити статичні Web-сторінки додатку «Linguevinchik» (тематика: вивчення іноземних слів) та перевірити коректне їх відображення на екранах різного розміру.

**Функціональні вимоги до інтерфейсу:**.
* Реєстрація користувача (поля: ім'я, email, стать, дата народження).
* Вхід до сайту (поля: email, пароль).
* Профіль користувача (дані у табличному вигляді).
* Сторінка «Про додаток» (емблема та опис).
* Робоча сторінка додатка: режими вивчення слів та контролю знань.
